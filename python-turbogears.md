# 一、安装 #

## 1.1 linux环境安装方式 ##


### 1.1.1 虚拟环境安装方式 ##

安装虚拟环境
	
	[root@Siffre ~]# pip install virtualenv
	
创建虚拟环境
	
	[root@Siffre ~]# virtualenv tgenv
	[root@Siffre ~]# . tgenv/bin/activate
	进入虚拟环境：
	(tgenv)[root@Siffre ~]#
	
安装TurboGears

	(tgenv)[root@Siffre ~]# pip  install  TurboGears
### 1.1.2 直接安装方式 ###
	[root@Siffre ~]# pip  install  TurboGears
	或
	easy_install TurboGears
